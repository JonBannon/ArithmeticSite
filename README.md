# ArithmeticSite

A Lean 4 formalization of the Connes-Consani Arithmetic Site, heavily using Claude Code. (This is an experiment to see
the degree to which the LLM can aid in development, and to help quickly stub out research projects for students.)

The project blueprint and API documentation are available at [https://jonbannon.github.io/ArithmeticSite/](https://jonbannon.github.io/ArithmeticSite/).
